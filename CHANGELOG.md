# Changelog público

## 0.1.0

Versão inicial de protótipo público.

- Terminal para Windows baseado no quardCRT
- SSH2, SFTP, Telnet, Serial e shell local
- Gerenciamento de sessões e grupos
- Importação de sessões
- Suporte inicial a automação de logon
- Suporte a prompts no terminal e keyboard-interactive para fluxos com Balabit
- SFTP simplificado com foco no servidor remoto
- Versão portable para testes
