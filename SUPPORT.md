# Suporte

Use este repositório público para reportar problemas e solicitar melhorias.

## Antes de abrir uma issue

Verifique se você consegue informar:

- versão do ZashCRT
- sistema operacional Windows
- tipo de conexão: SSH2, SFTP, Telnet, Serial ou shell local
- mensagem de erro exibida
- passos para reproduzir
- se envolve SecureCRT, Balabit, chave SSH ou script de logon

## Segurança

Não publique:

- senhas
- chaves privadas
- IPs internos sensíveis
- nomes reais de servidores corporativos
- scripts com credenciais
- prints contendo dados sensíveis

Ao enviar logs ou imagens, remova informações confidenciais antes.
