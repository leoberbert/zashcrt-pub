# ZashCRT

Terminal para Windows com SSH, SFTP, Telnet, Serial e shell local, com gerenciamento de sessões, importação de sessões, automação de logon e suporte a fluxos com Balabit.

Este é o repositório público do ZashCRT. Ele existe para concentrar documentação, site, issues, sugestões, roadmap, changelog público e suporte aos usuários.

O desenvolvimento principal do ZashCRT é mantido em repositório privado enquanto o projeto evolui. Este repositório público permite que usuários acompanhem o projeto, reportem problemas e solicitem melhorias sem expor o código-fonte principal.

## O que pode ser aberto em Issues

- Bugs e falhas de execução
- Problemas de conexão SSH, Telnet, Serial ou shell local
- Erros de SFTP, upload, download ou navegação remota
- Problemas com importação de sessões
- Solicitações de compatibilidade com Balabit
- Sugestões de UX e melhoria visual
- Pedidos de novas funcionalidades
- Dúvidas de configuração

## Recursos principais

- SSH2 com senha, chave e fluxos keyboard-interactive
- SFTP com navegação remota, download, upload e busca
- Telnet, Serial e shell local
- Gerenciamento de sessões e grupos
- Importação de sessões
- Automação de logon com ações esperadas/enviar e scripts
- Suporte a fluxos com Balabit
- Túneis SSH
- Versão portable para Windows
